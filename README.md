# VOX DRIVE

three.js の平面ワールドを、MagicaVoxel (.vox) 形式の車で走り回るゲームです。

- **Volvo** … プレイヤー操作車
- **Nissan 180SX / VW** … 街路を周回する AI 車(左側通行)
- **tree01 / tree02** … フィールドにランダム配置される樹木(当たり判定あり)

## 街路(自動生成)

- 縦横の格子道路 + たまに斜めの道路(シード値で決定的に生成)
- 道路は2種類: **片側二車線(計4車線・幅14m)** と **片側一車線(計2車線・幅8m)**
- 白線は日本方式: 外側線=実線、4車線の中央線=実線+車線境界=破線、2車線の中央線=破線
- 交差点内は白線なし(素のアスファルト)
- 区画(ブロック)には **2棟または4棟** の建物を配置。建物の前面は必ず道路側を向く
- 建物は現在プレースホルダーの箱。`js/main.js` の `BUILDING_VOX` に .vox ファイル
  (前面 = モデルの +Z)を追加すると差し替わる

## 遊び方

ブラウザのセキュリティ制限 (fetch) のため、ローカルサーバー経由で開いてください。

```sh
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 操作

| キー | 動作 |
| --- | --- |
| `A` | アクセル |
| `S` | ブレーキ |
| `↑` | シフトアップ |
| `↓` | シフトダウン |
| `←` `→` | ハンドル |
| `SPACE` | サイドブレーキ(ドリフト) |
| マウスドラッグ | カメラ視点 |
| ホイール | ズーム |
| `M` | サウンド ON/OFF |

ギアは `R / N / 1〜5` のマニュアル式。各ギアに最高速度があるので、上手にシフトチェンジして走ってください。

スピードに乗った状態で `SPACE` + ハンドルでテールが流れてドリフトします。滑っている間はスキッドマークとタイヤスモークが出ます。カウンターステアで姿勢をコントロールしてください。

サウンドは Web Audio API によるリアルタイム生成(音声ファイル不使用)。エンジン音はギアごとに音域が決まっており、回転数と共にピッチが上がります(1速は低音、上のギアほど高音域)。ドリフトや急ブレーキではタイヤのスキール音が鳴ります。

## 構成

- `index.html` — エントリ + HUD
- `js/vox.js` — 自作 .vox パーサー(SIZE/XYZI/RGBA + nTRN/nGRP/nSHP シーングラフ対応)。面カリング済みのマージ済みジオメトリを生成
- `js/main.js` — ゲーム本体(車の物理、AI、カメラ、樹木配置)
- `lib/three.min.js` — three.js r152(同梱)
- `vox/` — ボクセルモデル
